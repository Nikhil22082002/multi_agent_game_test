\# Multi-Agent Game Tester POC



This project demonstrates a proof-of-concept multi-agent testing system

for web-based puzzle and math games.



\## Features

\- Planner, Ranker, Executor, Analyzer agents

\- FastAPI backend

\- Artifact capture

\- JSON report generation

\- Minimal frontend UI



\## How to Run

1\. Install dependencies

2\. Run FastAPI backend

3\. Trigger tests via UI or API



\## Target Game

https://play.ezygamers.com/
